Это - оригинальные исходные коды плагина SuperWDX для Total Commander (http://www.tckb.ru/wiki/SuperWDX).
Выложено на github в надежде, что кто-то продолжит разработку.

Включены:
- Исходники плагина.
- Исходники утилиты настройки (для компиляции требуется модуль gspagecontrol).
- Пример работы с API плагина.
